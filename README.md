## ✨ Features

🖼️ **Import Custom Images** - Upload any photo from your device  
🔍 **Zoom & Resize** - Perfect positioning with intuitive controls  
🎨 **Multiple Filters** - Adjust tone and style with various filters  
✏️ **Custom Text** - Add any name and bounty amount  
💾 **Export as Image** - Download your poster anytime  
📱 **Mobile Friendly** - Works seamlessly on all devices  
⚡ **PWA Support** - Install as a progressive web app  
🎭 **Special Mode** - Discover the hidden feature!

## 🚀 Quick Start

### Try it Online
Visit the live demo: **[One Piece Wanted Poster Maker](https://tusharv2005.github.io/One-Piece-Wanted-Poster/)**

### Run Locally

```bash
# Clone the repository
git clone https://github.com/tusharv2005/One-Piece-Wanted-Poster.git

# Navigate to project directory
cd One-Piece-Wanted-Poster

# Install dependencies
pnpm install

# Start development server
pnpm run dev
```

The app will be available at `http://localhost:5173/`

## 🛠️ Tech Stack

- **Frontend Framework:** Vanilla TypeScript
- **Build Tool:** Vite
- **Styling:** CSS3
- **PWA:** Workbox
- **Package Manager:** pnpm

## 📦 Build for Production

```bash
# Build the project
pnpm run build

# Preview production build
pnpm run preview
```

## 🎯 How to Use

1. **Upload Photo** - Click to import an image from your device
2. **Adjust Position** - Zoom and resize to fit perfectly
3. **Apply Filters** - Choose from multiple tone adjustments
4. **Add Details** - Enter the character name and bounty
5. **Export** - Download your custom wanted poster!
